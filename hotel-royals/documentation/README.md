# Hotel Royals

Hotel Royals is a responsive hotel website template.

## Template features

- Mobile First strategy
- HTML5 & CSS3 responsive theme
- Clean design & layout
- Built with Bootstrap 4
- SEO Markup
- Responsive Images
- Testimonial Carousel
- 4 Pages
- Well-commented code
- Easy Setup
